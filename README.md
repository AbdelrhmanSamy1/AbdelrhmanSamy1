## Featured Project

### [Bookly REST API](https://github.com/AbdelrhmanSamy1/bookly-rest)
A production-ready bookstore REST API built with Spring Boot 3 and Java 21.

- JWT Authentication with Refresh Token Rotation & reuse detection
- Role-Based Access Control (ADMIN / USER)
- Dynamic filtering via JPA Specifications, pagination & sorting
- Dockerized — runs with a single `docker-compose up`
- Live Swagger UI: [bookly-rest-production.up.railway.app/swagger-ui.html](https://bookly-rest-production.up.railway.app/swagger-ui.html)
